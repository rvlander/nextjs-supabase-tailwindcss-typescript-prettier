# nextjs-supabase-tailwindcss-typescript-prettier

Sane start with nextjs, supabse, typescript and prettier as of March 2022

## Usage

Just download the archive of the last commit from github

Install dependencies:

```sh
nvm install
nvm use
npm install
```

Start your app:

```sh
npm run dev
```
